Proyecto Titanic: Machine Learning e Inteligencia Visual

Este proyecto implementa modelos supervisados para entender y predecir variables del histórico naufragio.

Ciclo de Vida del Proyecto (CRISP-ML)

  • 1. Comprensión del Negocio: Enfoque predictivo de supervivencia y costos de pasajes.
  • 2. Comprensión de Datos: Análisis del dataset histórico del Titanic.
  • 3. Preparación de Datos: Limpieza de nulos y codificación con Python.
  • 4. Modelado y Evaluación: Creación de algoritmos con Scikit-Learn.
