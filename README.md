# 🚢 Predicción de Supervivencia y Tarifas del Titanic

Este proyecto implementa un sistema predictivo dual utilizando algoritmos de aprendizaje supervisado (**Regresión Logística** y **Regresión Lineal**) sobre el histórico conjunto de datos del Titanic. El flujo de trabajo sigue rigurosamente la metodología **CRISP-ML**.

## 📊 Contenido del Repositorio
- `modelo_crisp_ml.ipynb`: Cuaderno Jupyter con las fases de análisis, limpieza de datos, entrenamiento y evaluación de los modelos.
- `app.py`: Aplicación interactiva desarrollada en **Streamlit** para simular escenarios en tiempo real.
- `index.html`: Landing Page explicativa que comunica el ciclo de vida del proyecto e integra conceptos visuales de **Realidad Aumentada (RA)**.
- `Realidad Aumentada/`: Carpeta que almacena los recursos visuales e imágenes generadas por IA.
